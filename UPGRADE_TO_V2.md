## 🚀 升级步骤（3 分钟完成）

### 步骤 1: 部署 V2 代码

```bash

# 部署（选择一种方式）

# 方式 B: 使用 Cloudflare Dashboard
# 1. 访问 https://dash.cloudflare.com
# 2. Workers & Pages → 选择你的 Worker
# 3. Quick Edit → 粘贴 src/worker.v2.js 的内容
# 4. Save and Deploy
```

## 🎬 新功能快速体验

### 功能 1: Bot 邀请链接

**体验流程**：

1. 私聊 Bot，发送 `/start`
2. 选择或创建一个视频库
3. 点击"添加转发目标"
4. **看到新按钮**：`🔗 点击添加到群组`
5. 点击按钮
6. **Telegram 弹出原生界面**，显示你的所有群组
7. 选择一个群组
8. ✅ Bot 自动加入并绑定成功！

**效果**：
```
Bot（在群组中）：
🎉 自动绑定成功！
📦 视频库: 测试库
🎯 该群组已自动添加为转发目标

Bot（私聊）：
✅ 目标已添加！
📦 视频库: 测试库
🎯 测试群组 (group)
```

---

### 功能 2: 群组快捷绑定

**体验流程**：

1. 确保 Bot 在目标群组中（如果不在，先添加）
2. 在群组中发送：`/bind 视频库名称`
3. ✅ 立即绑定成功！

**示例**：
```
在"搞笑视频群"中：
/bind 搞笑视频

Bot 响应：
🎉 绑定成功！
📦 视频库: 搞笑视频
🎯 该群组已添加为转发目标
现在会自动接收转发的内容！
```

**使用场景**：
- 管理多个群组时，无需切换到私聊
- 快速批量绑定
- 其他管理员也可以操作（需要视频库管理员权限）

---

### 功能 3: 备份频道

**体验流程**：

1. **创建备份频道**：
   - 在 Telegram 中创建新频道
   - 名称如："我的视频库-备份"
   - 设为私有频道（推荐）

2. **添加 Bot**：
   - 将 Bot 添加为频道管理员
   - 勾选"发布消息"权限

3. **配置到视频库**：
   - 私聊 Bot，选择视频库
   - 点击"💾 备份设置"
   - 转发备份频道的任意一条消息
   - ✅ 配置成功！

4. **测试自动备份**：
   - 开始转发模式
   - 发送任意内容
   - 检查备份频道，内容已自动保存 ✅

**效果**：
```
用户发送一张图片

Bot 转发到：
• 目标频道 1 ✅
• 目标频道 2 ✅
• 目标群组 3 ✅
• 备份频道（自动） ✅

Bot 响应：
✅ 已转发到 4/4 个目标
📦 我的视频库
📊 本次共 1 条
```

## 💡 使用技巧

### 技巧 1: 批量添加群组

使用 Bot 邀请链接快速添加多个群组：

```
点击"添加转发目标"
→ 点击"🔗 点击添加到群组"
→ 选择群组 1 → 完成
→ 再次点击"添加转发目标"
→ 点击"🔗 点击添加到群组"
→ 选择群组 2 → 完成
...
```

**效率**：每个群组 ~3 秒，10 个群组 30 秒搞定！

---

### 技巧 2: 在群组中统一绑定

如果 Bot 已在多个群组中，可以逐个发送 `/bind` 命令：

```
群组 1: /bind 视频库A
群组 2: /bind 视频库A
群组 3: /bind 视频库B
...
```

**效率**：复制粘贴命令，快速绑定！

---

### 技巧 3: 使用备份频道做数据仓库

**场景**：你有多个公开频道，但想集中保存所有内容

**方案**：
1. 创建一个私有备份频道"总备份"
2. 在多个视频库中都设置这个备份频道
3. 所有内容都会汇总到"总备份"

**好处**：
- 集中管理所有历史数据
- 方便检索和导出
- 即使删除视频库，数据依然存在

## 🔧 故障排除

### 问题 1: 点击 Bot 邀请链接没反应

**可能原因**：
- Bot username 未初始化
- 链接格式错误

**解决方案**：
```bash
# 检查 Webhook 信息
curl https://your-worker.workers.dev/webhook-info

# 确认 allowed_updates 包含 my_chat_member
# 如果没有，重新执行 /setup
curl https://your-worker.workers.dev/setup
```

---

### 问题 2: 群组快捷绑定提示"权限不足"

**可能原因**：
- 操作用户不是视频库管理员
- 视频库不存在

**解决方案**：
1. 确认视频库名称正确（大小写敏感）
2. 确认你是该视频库的创建者或管理员
3. 在私聊中用 `/start` 查看所有视频库

---

### 问题 3: 备份频道配置失败

**可能原因**：
- Bot 没有发布消息权限
- 频道 ID 错误
- 不是频道类型（群组不行）

**解决方案**：
1. 确认 Bot 是频道管理员
2. 确认 Bot 有"发布消息"权限
3. 确认是频道（channel），不是群组（group）
